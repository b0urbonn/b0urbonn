### 🔌 2. Windows on ARM (WOA) & Android Modding (Xiaomi Pad 5 "nabu")
I am highly active in the Windows on ARM modding scene, specializing in porting full desktops to mobile tablets like the **Xiaomi Pad 5 (nabu)**.
* 💻 **[Port-Windows-11-Xiaomi-Pad-5](https://github.com/b0urbonn/Port-Windows-11-Xiaomi-Pad-5)**: A comprehensive guide and installation script catalog for flashing Windows 11 to the Xiaomi Pad 5 (nabu).
* ⚙️ **[won-deployer-setup-app](https://github.com/b0urbonn/won-deployer-setup-app)**: Interactive deployment tool to automate downloading UEFI boot elements, setting up WOA Helpers, and flashing Magisk roots for tablet environments.
* 🌐 **[nabu](https://github.com/b0urbonn/nabu)**: Browser-based Web Fastboot toolkit that boots images (custom recovery) and flashes partitions directly from web interfaces.
* 🛠️ **[Auto-Installer-Forge](https://github.com/b0urbonn/Auto-Installer-Forge)**: Automated parser script targeting recovery ROM `payload.bin` archives, packaging them into Fastboot-flashable setups.
* 💾 **[WinInstaller](https://github.com/b0urbonn/WinInstaller)**: Standalone installer utility allowing users to deploy Windows on ARM directly from the device without needing an external PC.
* 🐧 **[Nabu_Linux_Installer](https://github.com/b0urbonn/Nabu_Linux_Installer)**: Open-source installer porting full Linux distributions onto the nabu device structure.
* 💿 **[woawin](https://github.com/b0urbonn/woawin)**: Windows 11 ARM64 downloader utility designed to fetch and prepare official Microsoft ESD/ISO compilation blocks.
* 📜 **[modified-playbooks](https://github.com/b0urbonn/modified-playbooks)**: Custom modified Atlas OS / AME Playbooks configured for optimal resource usage on Windows on ARM installations.
